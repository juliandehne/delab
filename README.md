# whowhatwhy
Open Source Project to bundle political opinions in a concise and validated fashion. Uses natural language processing to sort out inflammatory or overly positive comments. The branch delab is the temporary repository for the delab project of Uni GÃ¶ttingen.


# for programmers
You can start the project with python3:
- install the required packages with pip (in requirements.txt)
- run "python manage.py makemigrations"
- run "python manage.py migrate"
- run "python manage.py runserver"
